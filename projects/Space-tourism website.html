  • Space Tourism Multi-page Website

    Project Title: Space Tourism Website


    Overview:


    • The Challenge: Create a responsive multipage website for space tourism that adapts to different screen sizes and allows users to toggle between tabs to view different information.
    • Role: Lead Developer and Designer

    The Process:

    1. Research and Planning: Analyzed the project requirements from Frontend Mentor, planned the site layout, and identified the key features needed for an optimal user experience.
    2. Design: Created wireframes and mockups using Figma, focusing on a clean, user-friendly design that highlights the futuristic theme of space tourism.
    3. Development:
      • Built With: Semantic HTML5, CSS custom properties, Flexbox, CSS Grid, JavaScript
      • Responsive Design: Implemented a mobile-first approach to ensure the website looks great on all devices.
      • Interactive Elements: Added hover states and tab toggling functionality using JavaScript.

    Challenges and Solutions:

    • Challenge: Ensuring consistent layout and design across various screen sizes.
    • Solution: Utilized CSS Grid and Flexbox to create a flexible layout that adjusts smoothly.
    • Challenge: Implementing complex hover states for interactive elements.
    • Solution: Used CSS transitions and animations to achieve the desired effects.

    Results and Learnings:

    • Learnings: Enhanced my skills in CSS Grid and JavaScript, particularly in creating responsive designs and interactive elements.
    • Outcome: Delivered a fully functional, visually appealing space tourism website that meets all the project requirements.
